# FinerWorks Ruby Gem

A Ruby interface for the [FinerWorks Web API V2](http://api.finerworks.com/).


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'finerworks2'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install finerworks2
